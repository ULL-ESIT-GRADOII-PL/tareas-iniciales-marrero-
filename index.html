NodeJS

En primer lugar, para poder instalar el intérprete de Javascript NodeJS debemos descargar los ejecutables desde la web oficial. En este caso la instalación se hará para ordenadores con sistema operativo Windows. Tras descargar el fichero con extensión .msi solo tendrás que seguir los sencillos pasos que te indican en el instalador. Tras la instalación se puede ejecutar una terminal Node.js, aquí una captura:

Express

Express es un framework minimalista para NodeJS que se caracteriza por su velocidad. Para instalarlo tenemos que abrir la terminal que citábamos en el punto anterior y ejecutar el siguiente comando:

    $ npm install express --save

En este caso ya lo teníamos instalado y el resultado de la ejecución fue la siguiente:

Express
